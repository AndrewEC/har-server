Cloning
=======

.. toctree::
   :maxdepth: 4


To clone the project and the required submodules run::

    git clone --recurse-submodules https://github.com/AndrewEC/har-server.git

This will clone the entire project and initialize the two required submodules:

* `py-build-utils <https://github.com/AndrewEC/py-build-utils>`_
